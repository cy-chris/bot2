## Clone o projeto 

```bash
> git clone https://github.com/cy-chris/milenabot
```

## Instale as dependências:
Antes de executar o comando abaixo, verifique se você está no diretório do projeto que
você acabou de clonar!!

```bash
> cd milenabot
> bash install.sh
```

### para iniciar
```bash
> npm start
```
